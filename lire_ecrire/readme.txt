Ouverture Fichier :
	fopen(nomFichier,modeLecture): Sert a ouvrir le fichier.
Lecture Fichier :
	fgets(pointeurFichier): Recupére une ligne de caractères.
Ecriture Fichier :
	fputs(pointeurFichier,aEcrire): Ecris dans le fichier.
Fermeture Fichier :
	fclose(pointeurFichier): Fermer le fichier.
Deplacement Curseur :
	fseek(pointeurFichier,positionCurseur): Positionner le curseur.(sauf en cas d'ouverture en mode ajout:'a' ou 'a+').
